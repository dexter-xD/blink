Command-Line Usage

Start the server with the following command:

./bin/blink
  • -p, --port PORT Specify port number (default: 8080)
  • -s, --serve FILE Specify a custom HTML file to serve
  • -n, --no-templates Disable template processing
  • -h, --help Display this help message

Examples:

Default startup (port 8080):

./bin/blink

Use a different port:

./bin/blink -s my-page.html -p 8081

Serve a specific HTML file with templates disabled:

./bin/blink --serve my-page.html -n
